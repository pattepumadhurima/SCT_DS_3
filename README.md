# # Decision Tree Classifier for Bank Marketing Dataset

## Objective
Predict whether a customer will subscribe to a term deposit.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn
- Google Colab

## Steps Performed
1. Data Loading
2. Data Preprocessing
3. Label Encoding
4. Train-Test Split
5. Decision Tree Model Training
6. Model Evaluation
7. Visualization

## Results
The Decision Tree model successfully predicted customer deposits with good accuracy and identified important influencing features.
